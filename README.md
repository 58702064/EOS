简介
-------------------------------

区块链中文字幕/EOS 

是区块链中文字幕下一个EOS爱好者发起成立的翻译小组，致力从EOS技术文档的翻译、传播和推广。

欢迎更多的志愿者加入我们的团队。

任务组成
-------------------------------

- **选题**  负责选择合适的内容，并将原文转换为 markdown 格式，提交到。

- **译者**  负责从选题中选择内容进行翻译。

- **审核**  负责将初译的文章进行审核、润色、技术校对等工作。

- **发布** 负责将校对后的文章，排版进行发布。

组织构成
-------------------------------

- **专家组**  负责选题、指导、审核等工作
- **翻译组**  负责翻译相关技术文档
- **运营组**  负责传播、校核、技术支撑等工作
